# Skill Template

This is the template used by autoresearch to generate domain-specific standalone skills in the user's project.

## Template Variables

| Variable | Source | Example |
|----------|--------|---------|
| `{{SLUG}}` | Derived from goal | `test-coverage` |
| `{{GOAL}}` | User's natural language goal | `Increase test coverage to 80%` |
| `{{LANGUAGE}}` | Detected or confirmed | `python` |
| `{{STRATEGY}}` | From strategy selection | `coverage-improvement` |
| `{{TARGET_FILES}}` | From discovery | `tests/**/*.py` |
| `{{PROTECTED_FILES}}` | From discovery | `src/**/*.py, *.lock` |
| `{{EVAL_COMMAND}}` | From discovery | `pytest --cov=src` |
| `{{SCRIPT_EXT}}` | From language | `py` |

## Generated SKILL.md Template

```markdown
---
name: autoresearch-{{SLUG}}
description: >
  Autonomous optimization loop: {{GOAL}}.
  This skill was generated by autoresearch and runs an eval-driven
  improvement loop targeting {{TARGET_FILES}}.
---

# Autoresearch: {{GOAL}}

This is an autonomous optimization skill generated by autoresearch.
It runs an eval-driven loop that repeatedly improves code against
binary pass/fail metrics.

## Goal

{{GOAL}}

## Architecture

### Fixed Foundation (DO NOT MODIFY during loop)
- `eval.json` — Eval definitions and pass conditions
- `scripts/eval_runner.{{SCRIPT_EXT}}` — Eval execution script

### Agent Playground (modify freely)
- Target files: {{TARGET_FILES}}

### Protected Files (DO NOT MODIFY)
- {{PROTECTED_FILES}}
- This skill directory (except state.json and results/)

## Loop Protocol

Follow this protocol autonomously without pausing for user input.

### State Machine

INIT → ANALYZE → HYPOTHESIZE → MODIFY → EVALUATE → DECIDE → [ANALYZE | DONE]

### ANALYZE
1. Read the latest results (baseline or last iteration) from `results/`
2. Identify failing evals from eval.json
3. Read target files and error output
4. Check hypothesis log in state.json for previously tried approaches

### HYPOTHESIZE
1. Form a specific, testable hypothesis
2. Verify it hasn't been tried before (check state.json)
3. Log the hypothesis to state.json before making changes

### MODIFY
1. Verify all files are within target_files bounds
2. Verify no files are in protected_files
3. Make minimal, focused changes implementing the hypothesis
4. Verify changes parse/compile

### EVALUATE
1. Run: `{{SCRIPT_EXT}} <skill_path>/scripts/eval_runner.{{SCRIPT_EXT}}`
2. Collect pass/fail results for each eval
3. Save results to `results/iter-<NNN>.json`

### DECIDE
- **Improvement** (more evals pass): `git commit` with message format:
  `autoresearch({{SLUG}}): iter <N> — <hypothesis summary>`
- **Regression** (fewer evals pass): `git checkout -- <target_files>`
- **Neutral** (same results): `git checkout -- <target_files>`

### Termination
Stop when: all evals pass OR iteration >= max_iterations OR plateau detected.

## Strategy: {{STRATEGY}}

[Domain-specific strategy guidance inserted here based on strategy_hint]

## Safety Rules

1. NEVER modify eval.json, eval_runner, or this SKILL.md during the loop
2. ONLY modify files matching: {{TARGET_FILES}}
3. NEVER modify files matching: {{PROTECTED_FILES}}
4. Commit improvements, revert regressions — no exceptions
5. Log every hypothesis before modifying code
6. Stop at max_iterations even if goals aren't met
7. Stop if no progress for {{PLATEAU_THRESHOLD}} consecutive iterations
8. No interactive prompts during the loop
9. All work on experiment branch — never modify main/master
```

## Generated eval_runner Template (Python)

```python
#!/usr/bin/env python3
"""Eval runner for autoresearch-{{SLUG}}. DO NOT MODIFY during loop."""

import json
import re
import subprocess
import sys
import time
from pathlib import Path

SKILL_DIR = Path(__file__).parent.parent
EVAL_CONFIG = SKILL_DIR / "eval.json"
RESULTS_DIR = SKILL_DIR / "results"


def load_config():
    return json.loads(EVAL_CONFIG.read_text())


def run_eval(eval_def, timeout):
    start = time.time()
    try:
        result = subprocess.run(
            eval_def["command"],
            shell=True,
            capture_output=True,
            text=True,
            timeout=timeout,
        )
        duration = time.time() - start
        return {
            "exit_code": result.returncode,
            "stdout": result.stdout,
            "stderr": result.stderr,
            "duration_seconds": round(duration, 2),
            "timed_out": False,
        }
    except subprocess.TimeoutExpired:
        duration = time.time() - start
        return {
            "exit_code": -1,
            "stdout": "",
            "stderr": f"Timed out after {timeout}s",
            "duration_seconds": round(duration, 2),
            "timed_out": True,
        }


def check_pass_condition(condition, run_result):
    ctype = condition["type"]
    output = run_result["stdout"] + run_result["stderr"]

    if ctype == "exit_code_zero":
        return run_result["exit_code"] == 0

    if ctype == "output_contains":
        return condition["value"] in output

    if ctype == "output_not_contains":
        return condition["value"] not in output

    if ctype == "regex_match":
        match = re.search(condition["pattern"], output)
        if not match:
            return False
        try:
            value = float(match.group(condition.get("group", 1)))
        except (IndexError, ValueError):
            return False
        threshold = condition["threshold"]
        op = condition["operator"]
        if op == ">=": return value >= threshold
        if op == ">":  return value > threshold
        if op == "<=": return value <= threshold
        if op == "<":  return value < threshold
        if op == "==": return value == threshold
        if op == "!=": return value != threshold
        return False

    if ctype == "llm_judge":
        # LLM judge is handled by the agent inline, not by this script.
        # Output the data for the agent to evaluate.
        print(f"LLM_JUDGE_EVAL:{json.dumps({'output': output, 'criteria': condition['criteria']})}")
        return None  # Agent decides

    return False


def main():
    config = load_config()
    default_timeout = config["config"].get("timeout_per_eval_seconds", 120)
    RESULTS_DIR.mkdir(exist_ok=True)

    iteration = int(sys.argv[1]) if len(sys.argv) > 1 else 0
    results = []
    pass_count = 0
    fail_count = 0

    for eval_def in config["evals"]:
        timeout = eval_def.get("timeout_seconds", default_timeout)
        run_result = run_eval(eval_def, timeout)
        passed = check_pass_condition(eval_def["pass_condition"], run_result)

        result_entry = {
            "eval_id": eval_def["id"],
            "passed": passed,
            "output_summary": (run_result["stdout"] + run_result["stderr"])[:500],
            "duration_seconds": run_result["duration_seconds"],
            "timed_out": run_result["timed_out"],
        }

        # Extract numeric value for regex_match evals
        if eval_def["pass_condition"]["type"] == "regex_match":
            output = run_result["stdout"] + run_result["stderr"]
            match = re.search(eval_def["pass_condition"]["pattern"], output)
            if match:
                try:
                    result_entry["extracted_value"] = float(
                        match.group(eval_def["pass_condition"].get("group", 1))
                    )
                except (IndexError, ValueError):
                    pass

        results.append(result_entry)
        if passed:
            pass_count += 1
        elif passed is not None:
            fail_count += 1

    output = {
        "iteration": iteration,
        "timestamp": time.strftime("%Y-%m-%dT%H:%M:%SZ", time.gmtime()),
        "results": results,
        "pass_count": pass_count,
        "fail_count": fail_count,
        "total": len(config["evals"]),
    }

    # Write results file
    if iteration == 0:
        result_file = RESULTS_DIR / "baseline.json"
    else:
        result_file = RESULTS_DIR / f"iter-{iteration:03d}.json"
    result_file.write_text(json.dumps(output, indent=2) + "\n")

    # Print summary
    print(f"Evals: {pass_count}/{len(config['evals'])} passing")
    for r in results:
        status = "PASS" if r["passed"] else "FAIL"
        print(f"  [{status}] {r['eval_id']} ({r['duration_seconds']}s)")

    # Exit with appropriate code
    sys.exit(0 if pass_count == len(config["evals"]) else 1)


if __name__ == "__main__":
    main()
```

## Generated eval_runner Template (JavaScript)

```javascript
#!/usr/bin/env node
/** Eval runner for autoresearch-{{SLUG}}. DO NOT MODIFY during loop. */

const fs = require("fs");
const path = require("path");
const { execSync } = require("child_process");

const SKILL_DIR = path.resolve(__dirname, "..");
const EVAL_CONFIG = path.join(SKILL_DIR, "eval.json");
const RESULTS_DIR = path.join(SKILL_DIR, "results");

function loadConfig() {
  return JSON.parse(fs.readFileSync(EVAL_CONFIG, "utf8"));
}

function runEval(evalDef, timeout) {
  const start = Date.now();
  try {
    const output = execSync(evalDef.command, {
      timeout: timeout * 1000,
      encoding: "utf8",
      stdio: ["pipe", "pipe", "pipe"],
    });
    return {
      exit_code: 0,
      stdout: output || "",
      stderr: "",
      duration_seconds: ((Date.now() - start) / 1000).toFixed(2),
      timed_out: false,
    };
  } catch (err) {
    return {
      exit_code: err.status || -1,
      stdout: err.stdout || "",
      stderr: err.stderr || "",
      duration_seconds: ((Date.now() - start) / 1000).toFixed(2),
      timed_out: err.killed || false,
    };
  }
}

function checkPassCondition(condition, runResult) {
  const output = (runResult.stdout || "") + (runResult.stderr || "");

  switch (condition.type) {
    case "exit_code_zero":
      return runResult.exit_code === 0;
    case "output_contains":
      return output.includes(condition.value);
    case "output_not_contains":
      return !output.includes(condition.value);
    case "regex_match": {
      const match = output.match(new RegExp(condition.pattern));
      if (!match) return false;
      const value = parseFloat(match[condition.group || 1]);
      if (isNaN(value)) return false;
      const t = condition.threshold;
      const ops = { ">=": value >= t, ">": value > t, "<=": value <= t, "<": value < t, "==": value === t, "!=": value !== t };
      return ops[condition.operator] || false;
    }
    case "llm_judge":
      console.log(`LLM_JUDGE_EVAL:${JSON.stringify({ output, criteria: condition.criteria })}`);
      return null;
    default:
      return false;
  }
}

function main() {
  const config = loadConfig();
  const defaultTimeout = config.config.timeout_per_eval_seconds || 120;
  fs.mkdirSync(RESULTS_DIR, { recursive: true });

  const iteration = parseInt(process.argv[2] || "0", 10);
  const results = [];
  let passCount = 0;
  let failCount = 0;

  for (const evalDef of config.evals) {
    const timeout = evalDef.timeout_seconds || defaultTimeout;
    const runResult = runEval(evalDef, timeout);
    const passed = checkPassCondition(evalDef.pass_condition, runResult);

    const entry = {
      eval_id: evalDef.id,
      passed,
      output_summary: ((runResult.stdout || "") + (runResult.stderr || "")).slice(0, 500),
      duration_seconds: parseFloat(runResult.duration_seconds),
      timed_out: runResult.timed_out,
    };

    if (evalDef.pass_condition.type === "regex_match") {
      const output = (runResult.stdout || "") + (runResult.stderr || "");
      const match = output.match(new RegExp(evalDef.pass_condition.pattern));
      if (match) {
        const v = parseFloat(match[evalDef.pass_condition.group || 1]);
        if (!isNaN(v)) entry.extracted_value = v;
      }
    }

    results.push(entry);
    if (passed) passCount++;
    else if (passed !== null) failCount++;
  }

  const output = {
    iteration,
    timestamp: new Date().toISOString(),
    results,
    pass_count: passCount,
    fail_count: failCount,
    total: config.evals.length,
  };

  const filename = iteration === 0 ? "baseline.json" : `iter-${String(iteration).padStart(3, "0")}.json`;
  fs.writeFileSync(path.join(RESULTS_DIR, filename), JSON.stringify(output, null, 2) + "\n");

  console.log(`Evals: ${passCount}/${config.evals.length} passing`);
  for (const r of results) {
    console.log(`  [${r.passed ? "PASS" : "FAIL"}] ${r.eval_id} (${r.duration_seconds}s)`);
  }

  process.exit(passCount === config.evals.length ? 0 : 1);
}

main();
```

## Generated eval_runner Template (Shell)

```bash
#!/usr/bin/env bash
# Eval runner for autoresearch-{{SLUG}}. DO NOT MODIFY during loop.
set -euo pipefail

SKILL_DIR="$(cd "$(dirname "$0")/.." && pwd)"
EVAL_CONFIG="$SKILL_DIR/eval.json"
RESULTS_DIR="$SKILL_DIR/results"
ITERATION="${1:-0}"

mkdir -p "$RESULTS_DIR"

# Requires jq for JSON parsing
if ! command -v jq &>/dev/null; then
  echo "ERROR: jq is required but not installed"
  exit 2
fi

PASS_COUNT=0
FAIL_COUNT=0
TOTAL=$(jq '.evals | length' "$EVAL_CONFIG")
RESULTS="[]"

for i in $(seq 0 $((TOTAL - 1))); do
  EVAL_ID=$(jq -r ".evals[$i].id" "$EVAL_CONFIG")
  COMMAND=$(jq -r ".evals[$i].command" "$EVAL_CONFIG")
  COND_TYPE=$(jq -r ".evals[$i].pass_condition.type" "$EVAL_CONFIG")
  TIMEOUT=$(jq -r ".evals[$i].timeout_seconds // .config.timeout_per_eval_seconds // 120" "$EVAL_CONFIG")

  START=$(date +%s%N)
  OUTPUT=""
  EXIT_CODE=0
  TIMED_OUT="false"

  if OUTPUT=$(timeout "${TIMEOUT}s" bash -c "$COMMAND" 2>&1); then
    EXIT_CODE=0
  else
    EXIT_CODE=$?
    if [ "$EXIT_CODE" -eq 124 ]; then
      TIMED_OUT="true"
    fi
  fi

  DURATION=$(( ($(date +%s%N) - START) / 1000000 ))
  DURATION_SEC=$(echo "scale=2; $DURATION / 1000" | bc)

  PASSED="false"
  case "$COND_TYPE" in
    exit_code_zero)
      [ "$EXIT_CODE" -eq 0 ] && PASSED="true"
      ;;
    output_contains)
      VALUE=$(jq -r ".evals[$i].pass_condition.value" "$EVAL_CONFIG")
      echo "$OUTPUT" | grep -qF "$VALUE" && PASSED="true"
      ;;
    output_not_contains)
      VALUE=$(jq -r ".evals[$i].pass_condition.value" "$EVAL_CONFIG")
      echo "$OUTPUT" | grep -qF "$VALUE" || PASSED="true"
      ;;
    regex_match)
      PATTERN=$(jq -r ".evals[$i].pass_condition.pattern" "$EVAL_CONFIG")
      GROUP=$(jq -r ".evals[$i].pass_condition.group // 1" "$EVAL_CONFIG")
      OP=$(jq -r ".evals[$i].pass_condition.operator" "$EVAL_CONFIG")
      THRESHOLD=$(jq -r ".evals[$i].pass_condition.threshold" "$EVAL_CONFIG")
      EXTRACTED=$(echo "$OUTPUT" | grep -oP "$PATTERN" | head -1 | grep -oP '\d+\.?\d*' | sed -n "${GROUP}p")
      if [ -n "$EXTRACTED" ]; then
        PASSED=$(echo "$EXTRACTED $OP $THRESHOLD" | bc -l | grep -q 1 && echo "true" || echo "false")
      fi
      ;;
    llm_judge)
      echo "LLM_JUDGE_EVAL: eval_id=$EVAL_ID"
      PASSED="null"
      ;;
  esac

  if [ "$PASSED" = "true" ]; then
    ((PASS_COUNT++)) || true
    STATUS="PASS"
  elif [ "$PASSED" != "null" ]; then
    ((FAIL_COUNT++)) || true
    STATUS="FAIL"
  else
    STATUS="JUDGE"
  fi

  echo "  [$STATUS] $EVAL_ID (${DURATION_SEC}s)"

  SUMMARY=$(echo "$OUTPUT" | head -c 500)
  RESULTS=$(echo "$RESULTS" | jq \
    --arg id "$EVAL_ID" \
    --argjson passed "$PASSED" \
    --arg summary "$SUMMARY" \
    --argjson duration "$DURATION_SEC" \
    --argjson timed_out "$TIMED_OUT" \
    '. + [{"eval_id": $id, "passed": $passed, "output_summary": $summary, "duration_seconds": $duration, "timed_out": $timed_out}]')
done

echo "Evals: $PASS_COUNT/$TOTAL passing"

TIMESTAMP=$(date -u +"%Y-%m-%dT%H:%M:%SZ")
OUTPUT_JSON=$(jq -n \
  --argjson iter "$ITERATION" \
  --arg ts "$TIMESTAMP" \
  --argjson results "$RESULTS" \
  --argjson pass "$PASS_COUNT" \
  --argjson fail "$FAIL_COUNT" \
  --argjson total "$TOTAL" \
  '{"iteration": $iter, "timestamp": $ts, "results": $results, "pass_count": $pass, "fail_count": $fail, "total": $total}')

if [ "$ITERATION" -eq 0 ]; then
  echo "$OUTPUT_JSON" > "$RESULTS_DIR/baseline.json"
else
  printf -v PADDED "%03d" "$ITERATION"
  echo "$OUTPUT_JSON" > "$RESULTS_DIR/iter-${PADDED}.json"
fi

[ "$PASS_COUNT" -eq "$TOTAL" ] && exit 0 || exit 1
```

## Generated loop_tracker Template (Python)

```python
#!/usr/bin/env python3
"""Loop state tracker for autoresearch-{{SLUG}}. Manages the optimization loop state machine."""

import json
import sys
import time
from pathlib import Path

SKILL_DIR = Path(__file__).parent.parent
STATE_FILE = SKILL_DIR / "state.json"

VALID_STATES = ("init", "analyze", "hypothesize", "modify", "evaluate", "decide", "done")

TRANSITIONS = {
    "init": {"analyze"},
    "analyze": {"hypothesize", "done"},
    "hypothesize": {"modify"},
    "modify": {"evaluate"},
    "evaluate": {"decide"},
    "decide": {"analyze", "done"},
    "done": set(),
    None: {"init"},
}


def load_state():
    if STATE_FILE.exists():
        return json.loads(STATE_FILE.read_text())
    return {
        "status": "not_started",
        "state": None,
        "iteration": 0,
        "baseline": None,
        "current": None,
        "best": None,
        "hypotheses": [],
        "plateau_counter": 0,
        "started_at": None,
        "last_updated": None,
    }


def save_state(state):
    state["last_updated"] = time.strftime("%Y-%m-%dT%H:%M:%SZ", time.gmtime())
    STATE_FILE.write_text(json.dumps(state, indent=2) + "\n")


def cmd_status(state):
    print(f"Status:    {state['status']}")
    print(f"State:     {state['state'] or 'not started'}")
    print(f"Iteration: {state['iteration']}")
    if state["current"]:
        print(f"Passing:   {state['current']['pass_count']}/{state['current']['total']}")
    if state["best"]:
        print(f"Best:      {state['best']['pass_count']}/{state['best']['total']} (iter {state['best']['iteration']})")
    print(f"Plateau:   {state['plateau_counter']}")
    print(f"Hypotheses tried: {len(state['hypotheses'])}")


def cmd_transition(state, target):
    if target not in VALID_STATES:
        print(f"ERROR: Invalid state '{target}'. Must be one of: {', '.join(VALID_STATES)}")
        sys.exit(1)

    current = state["state"]
    allowed = TRANSITIONS.get(current, {"init"})

    if target not in allowed:
        print(f"WARNING: Transition {current or 'none'} -> {target} is not standard.")
        print(f"  Expected: {', '.join(sorted(allowed))}")

    state["state"] = target
    if target == "init":
        state["status"] = "running"
        state["started_at"] = time.strftime("%Y-%m-%dT%H:%M:%SZ", time.gmtime())
    elif target == "done":
        state["status"] = state.get("done_reason", "complete")
        state["completed_at"] = time.strftime("%Y-%m-%dT%H:%M:%SZ", time.gmtime())
    elif target == "analyze":
        state["iteration"] += 1

    save_state(state)
    print(f"[{target.upper()}] iteration {state['iteration']}")
    return state


def cmd_hypothesis(state, hypothesis, target_eval, target_files):
    entry = {
        "iteration": state["iteration"],
        "hypothesis": hypothesis,
        "target_eval": target_eval,
        "target_files": target_files,
        "timestamp": time.strftime("%Y-%m-%dT%H:%M:%SZ", time.gmtime()),
        "result": None,
    }
    state["hypotheses"].append(entry)
    save_state(state)
    print(f"Logged hypothesis for iteration {state['iteration']}: {hypothesis[:80]}")


def cmd_result(state, pass_count, fail_count, total, committed):
    state["current"] = {"pass_count": pass_count, "fail_count": fail_count, "total": total}

    if committed:
        state["plateau_counter"] = 0
        if not state["best"] or pass_count > state["best"]["pass_count"]:
            state["best"] = {**state["current"], "iteration": state["iteration"]}
        if state["hypotheses"]:
            state["hypotheses"][-1]["result"] = "committed"
    else:
        state["plateau_counter"] += 1
        if state["hypotheses"]:
            state["hypotheses"][-1]["result"] = "reverted"

    save_state(state)
    status = "COMMIT" if committed else "REVERT"
    print(f"[{status}] {pass_count}/{total} passing (plateau: {state['plateau_counter']})")


def cmd_reset():
    if STATE_FILE.exists():
        STATE_FILE.unlink()
        print("State reset.")
    else:
        print("No state file found.")


def main():
    if len(sys.argv) < 2:
        print("Usage: loop_tracker.py <status|transition|hypothesis|result|reset> [args]")
        sys.exit(1)

    command = sys.argv[1]
    state = load_state()

    if command == "status":
        cmd_status(state)
    elif command == "transition":
        if len(sys.argv) < 3:
            print("ERROR: transition requires a state argument")
            sys.exit(1)
        cmd_transition(state, sys.argv[2])
    elif command == "hypothesis":
        if len(sys.argv) < 5:
            print("ERROR: hypothesis requires: <text> <target_eval> <target_files>")
            sys.exit(1)
        cmd_hypothesis(state, sys.argv[2], sys.argv[3], sys.argv[4].split(","))
    elif command == "result":
        if len(sys.argv) < 6:
            print("ERROR: result requires: <pass_count> <fail_count> <total> <committed:true|false>")
            sys.exit(1)
        cmd_result(state, int(sys.argv[2]), int(sys.argv[3]), int(sys.argv[4]), sys.argv[5] == "true")
    elif command == "reset":
        cmd_reset()
    else:
        print(f"ERROR: Unknown command '{command}'")
        sys.exit(1)


if __name__ == "__main__":
    main()
```
